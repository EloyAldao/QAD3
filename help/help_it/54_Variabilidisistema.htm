Variabili di sistema

Le variabili di Sistema sono delle impostazioni che controllano il comportamento di alcuni comandi. Possono essere di tipo intero, reale, carattere, booleano or colori RGB (es. “#FF0000”). Sono salvate e caricate nel file QAD.INI della cartella del progetto corrente se esiste oppure nel file QAD.INI situato nella cartella di installazione.
