Dimensioning

Dimension style is a set of properties that determine the appearance of dimensions. These properties are stored in files with the extension .dim and are loaded on QAD startup or on loading a project. Dimension files must be saved in the current project folder or in the QAD installation folder (i.e. in windows 8 "C:\users\current user\.qgis2\python\plugins\qad                ").

QAD stores the elements constituting a dimension in 3 different layers::

·         Text layer for storing dimension text

·         Symbol layers to store punctual dimension objects (dimension points, arrow symbols ...)

·         Linear layers to store linear dimension objects (dimension line, extension lines ...)
